# Domains

The benchmark covers 22 educational domains across three categories.

## STEM Domains

| Domain | Description |
|--------|-------------|
| **Calculus** | Largest domain (325 concepts). Derivatives, integrals, limits, series. |
| **Biology** | Cell biology, genetics, ecology, evolution. |
| **Genetics** | Mendelian genetics, molecular genetics, genomics. |
| **Bioinformatics** | Sequence analysis, structural biology, computational methods. |
| **Statistics** | Probability, distributions, hypothesis testing, regression. |
| **Quantum Computing** | Qubits, gates, algorithms, error correction. |
| **Circuits** | Electronic components, circuit analysis, digital logic. |
| **Geometry** | Euclidean geometry, proofs, transformations, measurement. |
| **Ecology** | Ecosystems, populations, biodiversity, conservation. |
| **Moss** | Bryophyte biology, morphology, reproduction, ecology. |
